Shiori / Settings
Language
Display language for the app.
Gallery cards
Quick actions
Controls when the download, translate, export, and delete buttons appear on gallery cards.
App-language flag
Cards show a small flag for each of a gallery's languages. Hide the flag that matches the app's current language, or show a flag for every language.
Reader
Study mode — original
How a bubble shows the original text in Study mode. Image keeps the untouched page and a click reveals the translation; Text overlays the original as selectable text on the cleaned page, and a click cycles between original and translation.
Original text font
The typeface used when the original text is shown as text in Study mode.
Study mode — translation
How a bubble shows the translation in Study mode. Image layers reproduce the translated typesetting exactly; Text renders the translation as selectable text. Pages without stored image layers always use text.
Furigana
In Study mode's text display, show kana readings above the kanji of the original text. Only galleries with a Japanese language tag show furigana.
Chapter overview
A series shows a chapter overview page before the reader. Skip it to open the first chapter directly (the chapter bar is still available inside the reader).
Scroll strip flow
Whether the scroll strip runs the whole series as one continuous strip or one chapter at a time. Thumbnails follow: series-wide when continuous, per-chapter otherwise.
Chapter transitions
Mark the divide between chapters — an inline divider in the scroll strip, a dedicated transition page in single/double modes. The per-chapter strip always keeps its end-of-chapter card.
Translation
Translation server
Base URL of your self-hosted manga-image-translator server. The "Translate" button on a gallery sends each cached page here and stores a translated copy alongside the original.
○ Checking…
Server access token
Only needed if the server requires one (shared or remote servers). Leave empty for a local server.
Study mode generation
What a translation stores for Study mode (revealing one bubble at a time in the reader). Off is the fastest; Text only adds each bubble's original and translated text; Text and image also stores the exact typeset image layers.
Library Backup
Export backup
Lightweight metadata-only backup (.shi — tags, titles, source links) or a full backup of the entire library including images (.shioridb) for moving between browsers or machines. You'll be asked which.
Import backup
Restore a .shi or .shioridb backup into this library. The format is detected automatically.
Stats
Total Disk Writes
Total image data ever written to IndexedDB across all sessions. Counts CDN downloads and local CBZ imports.
